# ScriptGen - AI Script Generator

## Visual Theme & Atmosphere
- Modern, clean, developer-focused
- Dark theme cu accente subtile
- Minimalist dar functional

## Color Palette
- Background: #0D0D0D (near black)
- Surface: #1A1A1A (cards)
- Border: #2A2A2A
- Primary: #10B981 (emerald green)
- Primary hover: #059669
- Text primary: #FAFAFA
- Text secondary: #A1A1AA
- Accent: #6366F1 (indigo pentru highlights)

## Typography
- Font: Geist (sans-serif), fallback system-ui
- Headings: 700 weight
- Body: 400 weight
- Code: JetBrains Mono, monospace

## Layout
- Max-width: 800px, centered
- Padding: 24px
- Card-based interface
- Spacing: 16px base unit

## Components
- Input: dark bg, border, focus ring emerald
- Button: emerald bg, white text, rounded
- Output: code block styling cu syntax highlighting look
- Category pills: clickable tags pentru diferite tipuri de script-uri

## Animations
- Smooth transitions (200ms)
- Fade in pentru output
